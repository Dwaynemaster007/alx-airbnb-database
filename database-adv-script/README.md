# Advanced Database Queries for ALX Airbnb

## Project: Unleashing Advanced Querying Power

This project is part of the ALX Airbnb Database Module, focusing on advanced SQL concepts, query optimization, and performance tuning. The `joins_queries.sql` file contains solutions for the initial tasks of writing complex queries using different types of joins.

### Files
* `joins_queries.sql`: Contains SQL queries demonstrating INNER JOIN, LEFT JOIN, and a workaround for FULL OUTER JOIN.

### How to Use
1.  Connect to your database client.
2.  Navigate to the `database-adv-script` directory.
3.  Execute the queries in `joins_queries.sql` against the corresponding Airbnb database schema.
